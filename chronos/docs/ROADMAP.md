# Chronos Roadmap - Future Functional Improvements

## Short-term (Next Release)
1. **Complete Dashboard**:
   - Finish daily digest, focus clock integration.
   - Interactive task quick-add, sub-task inline editing.
2. **Timeline Enhancements**:
   - Drag-and-drop rescheduling.
   - Calendar view (day/week/month).
   - Filter by priority/project/goal/tags.
3. **Focus Mode Polish**:
   - Full-screen timer with sounds/alarms.
   - Session history and streaks.
   - Break timers (Pomodoro 25/5).
4. **Settings**:
   - Theme customization, notification prefs.
   - Data export/import (JSON/CSV).
5. **Notifications**:
   - Due date reminders via flutter_local_notifications.
   - Launch at startup (desktop).

## Medium-term
1. **Search & Filters**:
   - Global search across tasks/goals.
   - Advanced filters (date range, status, tags).
2. **Analytics**:
   - Charts: Productivity trends, time spent per project.
   - Weekly/monthly reports via DigestSnapshots.
3. **Integrations**:
   - Calendar sync (Google Calendar, iCal export).
   - Cloud sync (Firebase/Supabase).
4. **Collaboration**:
   - Share projects/goals.
   - Multi-user tasks (basic).
5. **Mobile Optimizations**:
   - Gesture navigation.
   - Offline handling polish.

## Long-term (AI & Advanced)
1. **AI Features**:
   - Task suggestions based on habits.
   - Smart prioritization (Eisenhower matrix).
   - Natural language task creation.
2. **Customization**:
   - Plugin system for views/widgets.
   - Custom RRULE editors.
3. **Performance**:
   - Pagination for large datasets.
   - Web PWA support.
4. **Accessibility**:
   - Full WCAG compliance.
   - Voice commands.

## Tech Debt & Refactors
- Migrate to latest Drift/Flutter versions.
- Add integration tests.
- Code generation automation (Drift).
- i18n full support.

*Priorities based on user feedback and MVP completion.*
